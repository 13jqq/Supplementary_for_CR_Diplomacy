# CR²: Consistency-Regularized Coopetitive Reasoning for Strategic Decision-Making in No-Press Diplomacy

This repository contains the supplementary code for the paper:

**CR²: Consistency-Regularized Coopetitive Reasoning for Strategic Decision-Making in No-Press Diplomacy**

The codebase is built on top of the Cicero / FairDiplomacy framework and preserves the original project structure. We add the implementation of the proposed consistency-regularized coopetitive reasoning method for strategic decision-making in no-press Diplomacy.

For more details about the original Cicero codebase, environment setup, model files, and general usage, please refer to [readme_cicero.md](readme_cicero.md).
